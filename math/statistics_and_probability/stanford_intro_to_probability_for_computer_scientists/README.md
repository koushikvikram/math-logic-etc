## Notes

### Counting, Probability, Inference, Machine Learning

One compact way to relate the course themes:

* **Counting**: The discipline of enumerating possible outcomes before assigning probabilities.
* **Probability**: The language for quantifying uncertainty over those outcomes.
* **Random variables**: The bridge from outcomes to numeric quantities we can analyze.
* **Inference**: The process of learning from observed data under uncertainty.
* **Machine learning**: A major application area where probabilistic models support prediction and decision-making.

In other words: counting defines the space; probability weighs the space; random variables summarize the space; inference updates beliefs from data; machine learning applies those updates to real prediction tasks.

#### Computer science example

* **Counting**: Count possible hashes, paths, samples, or assignments.
* **Probability**: Model collisions, errors, uncertainty, or randomized behavior.
* **Inference**: Estimate unknown parameters from observed data.
* **Machine learning**: Use those estimates to classify, predict, rank, or decide.

If a model performs well on average but fails for a subpopulation, that behavior points back to the probability model, inference assumptions, and fairness analysis.

#### Conceptual chain

You can summarize the course arc like this:

> Counting → probability → random variables → inference → machine learning → responsible probabilistic reasoning

The course is about learning to reason with uncertainty precisely enough to analyze algorithms, data, and models.
