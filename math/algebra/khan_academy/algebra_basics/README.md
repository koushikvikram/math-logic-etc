5. Graphing the slope-intercept form

#### Graphing Lines with Integer Slopes

Let's graph the line:

```text
y = 2x + 3
```

Recall the general **slope-intercept form** of a line:

```text
y = mx + b
```

Where:

- **m** = slope
- **b** = y-intercept

For the equation:

```text
y = 2x + 3
```

- **Slope (m)** = **2**
- **Y-intercept (b)** = **3**

So the line crosses the y-axis at:

```text
(0, 3)
```

##### Step 1: Plot the y-intercept

The first point is:

```text
(0, 3)
```

##### Step 2: Use the slope

The slope is **2**, which can be written as:

```text
2 = 2/1
```

This means:

- Move **1 unit to the right** (increase x by 1)
- Move **2 units up** (increase y by 2)

Starting from **(0, 3)**:

```text
x = 0 + 1 = 1
y = 3 + 2 = 5
```

This gives the second point:

```text
(1, 5)
```

##### Why does this work?

A slope of **2** means:

> Every time **x** increases by **1**, **y** increases by **2**.

Starting from:

```text
(0, 3)
```

Increase:

- x by **1**
- y by **2**

Result:

```text
(0 + 1, 3 + 2) = (1, 5)
```

##### Points on the Line

| Point | How it was found |
|-------|-------------------|
| (0, 3) | y-intercept |
| (1, 5) | Move right 1, up 2 using the slope |

Once these two points are plotted, draw a straight line through them.

#### Graphing Lines with a Fractional Slope

Let's graph the line:

```text
y = (2/3)x + 1
```

Recall the **slope-intercept form**:

```text
y = mx + b
```

Where:

- **m** = slope
- **b** = y-intercept

For the equation:

```text
y = (2/3)x + 1
```

- **Slope (m)** = **2/3**
- **Y-intercept (b)** = **1**

So the line passes through:

```text
(0, 1)
```

##### Step 1: Plot the y-intercept

The first point is:

```text
(0, 1)
```

##### Step 2: Use the slope

The slope is:

```text
2/3
```

This means:

- Move **3 units to the right** (increase x by 3)
- Move **2 units up** (increase y by 2)

Starting from **(0, 1)**:

```text
x = 0 + 3 = 3
y = 1 + 2 = 3
```

This gives the second point:

```text
(3, 3)
```

##### Why not move right 1 and up 2/3?

Technically, the point:

```text
(1, 1⅔)
```

is also on the line.

However, points with fractional coordinates are harder to plot accurately by hand. Instead, it's much easier to use the slope as a **ratio** and move by whole numbers.

##### Understanding the Slope

Remember that slope is defined as:

```text
Slope = Change in y / Change in x
```

For this line:

```text
2/3 = Change in y / Change in x
```

If we choose:

```text
Change in x = 3
```

then:

```text
Change in y = 2
```

So:

- Increase x by **3**
- Increase y by **2**

This lands exactly on another point with integer coordinates.

##### General Rule for Fractional Slopes

For a slope:

```text
a/b
```

- Increase **x** by **b** units.
- Increase **y** by **a** units.

Equivalently:

| Slope | Move in x | Move in y |
|-------|----------:|----------:|
| 2/3 | +3 | +2 |
| 5/4 | +4 | +5 |
| 3/2 | +2 | +3 |
| 7/5 | +5 | +7 |

##### Points on the Line

| Point | How it was found |
|-------|-------------------|
| (0, 1) | y-intercept |
| (3, 3) | Move right 3, up 2 using the slope |

Once these two points are plotted, draw a straight line through them.
