# 1. Big Picture

Arithmetic works with known numbers.

```text
3 + 5 = 8
```

Algebra works with **unknown or changing quantities**.

```text
3x + 5
```

The letter **x** is called a **variable**.

A variable represents a number whose value may be unknown or may change.

Instead of solving one problem, algebra lets us solve **an entire family of problems**.

Example

```text
Distance

= Speed × Time
```

Instead of

```text
60 × 2 = 120 km
```

we write

```text
d = st
```

Now the same formula works for every speed and every time.

This is the power of variables.

---

# 2. Why This Exists

Imagine calculating monthly cloud costs.

January

```text
$2,300
```

February

```text
$2,700
```

March

```text
$2,450
```

Writing separate equations for every month quickly becomes impossible.

Instead we introduce a variable.

```text
C

= monthly cloud cost
```

Now we can write

```text
C = Storage + Compute + Networking
```

One equation describes infinitely many months.

Variables allow mathematics to describe **patterns**, not just individual cases.

# 3. First Principles

## What is a Variable?

A variable is a symbol representing a quantity.

Example

```text
x
```

could represent

```text
5
```

today,

and

```text
12
```

tomorrow.

---

## What is a Constant?

A constant never changes.

Examples

```text
5

12

π

100
```

---

## What is a Coefficient?

The number multiplying the variable.

Example

```text
7x
```

Coefficient

```text
7
```

Variable

```text
x
```

---

## What is a Term?

A term is one piece of an expression separated by

```text
+

or

−
```

Example

```text
3x + 7 − 2y
```

Terms are

```text
3x

7

−2y
```

---

## What are Like Terms?

Like terms have

**exactly the same variable part**.

Examples

```text
3x

5x

−8x
```

are like terms.

Examples

```text
3x

3y
```

are **not**.

Examples

```text
x

x²
```

are also **not**.

The variable **and its exponent** must match.

---

# 4. Core Definitions & Formulas

## General Form

```text
Coefficient × Variable
```

Example

```text
8x
```

---

## Combining Like Terms

```text
ax + bx

= (a+b)x
```

---

## Different Variables

```text
3x + 5y
```

Cannot simplify.

---

## Different Exponents

```text
4x + 7x²
```

Cannot simplify.

---

## Constants

```text
4 + 7

=11
```

Constants combine only with constants.

# 5. Intuition

Think about money.

Suppose you have

```text
3 dollars

+

5 dollars
```

You can combine them.

```text
8 dollars
```

Now suppose you have

```text
3 dollars

+

5 euros
```

Can you combine them?

No.

They represent different quantities.

Variables work exactly the same way.

```text
3x

+

5x
```

Both represent the same "unit."

They combine.

```text
8x
```

But

```text
3x

+

5y
```

cannot combine because they represent different quantities.

Another example.

Cloud infrastructure.

```text
3 CPUs

+

5 CPUs

=8 CPUs
```

But

```text
3 CPUs

+

5 GB RAM
```

cannot be added.

Different resources.

Different variables.

# 6. Engineering Insight

Programming languages use variables everywhere.

Python

```python
requests = 250
errors = 5
success = requests - errors
```

The symbols

```text
requests

errors
```

are variables.

Algebra gives us the mathematical language behind programming.

Machine learning also relies heavily on variables.

Example

```text
Loss = f(weights)
```

The weights change during training.

The variables represent quantities that evolve over time.

# 7. Step-by-Step Method

## Combining Like Terms

Step 1

Identify every term.

Example

```text
4x + 7 − 3x + 5
```

Terms

```text
4x

7

−3x

5
```

---

Step 2

Group like terms.

```text
4x

−3x
```

and

```text
7

5
```

---

Step 3

Add coefficients.

```text
4−3

=1
```

---

Step 4

Combine constants.

```text
7+5

=12
```

---

Final answer

```text
x +12
```

# 10. Common Mistakes

## Mistake 1

Combining different variables.

Incorrect

```text
3x +4y

=7xy
```

Correct

```text
3x +4y
```

No simplification.

---

## Mistake 2

Combining different exponents.

Incorrect

```text
2x +3x²

=5x³
```

Correct

```text
2x +3x²
```

---

## Mistake 3

Forgetting the coefficient.

```text
x
```

actually means

```text
1x
```

Example

```text
x+5x

=6x
```

---

## Mistake 4

Dropping negative signs.

Incorrect

```text
5x−8x

=3x
```

Correct

```text
−3x
```

---

## Mistake 5

Thinking variables are labels instead of quantities.

Variables represent numbers.

The rules of arithmetic still apply to them.

# 11. Pattern Recognition

The first question to ask is:

> **Can these terms represent the same quantity?**

If the answer is **yes**, combine them.

If the answer is **no**, leave them separate.

---

## Type 1: Same Variable

Example

```text
4x + 9x
```

Ask:

> Do they have the same variable?

Yes.

```text
(4+9)x

=13x
```

---

## Type 2: Different Variables

Example

```text
4x +9y
```

Ask:

> Same variable?

No.

Cannot combine.

---

## Type 3: Same Variable, Different Exponents

Example

```text
7x

+

2x²
```

Variables match.

Exponents do not.

Cannot combine.

Think of

```text
7 apples

+

2 apple boxes
```

Different kinds of quantities.

---

## Type 4: Constants

Example

```text
7

−3

+12
```

Constants combine.

```text
16
```

---

## Type 5: Mixed Expressions

Example

```text
5x

+

7

−2x

+

9
```

Group first.

```text
(5x−2x)

+

(7+9)
```

Then simplify.

```text
3x+16
```

---

## Type 6: Hidden Coefficients

Remember

```text
x

=1x
```

Example

```text
x

+

6x

−2x
```

```text
1+6−2

=5
```

Answer

```text
5x
```

# 15. Historical Insight

The use of letters to represent unknown quantities was popularized in the late 16th century by the French mathematician François Viète.

Later, René Descartes standardized the convention of using:

* **x**, **y**, and **z** for unknowns.
* **a**, **b**, and **c** for known constants.

Before symbolic algebra, mathematicians wrote equations entirely in words, making even simple problems cumbersome.

Variables transformed algebra from a collection of arithmetic tricks into a general language for describing patterns, relationships, and physical laws.

# 17. Summary

Variables represent changing or unknown quantities.

Like terms have:

* the same variable, and
* the same exponent.

Only like terms can be combined.

Core rule:

```text
ax+bx

=(a+b)x
```

Think of variables as **units**.

You can combine

```text
3 CPUs

+

5 CPUs
```

but not

```text
3 CPUs

+

5 GB RAM
```

The same principle applies to algebra.
