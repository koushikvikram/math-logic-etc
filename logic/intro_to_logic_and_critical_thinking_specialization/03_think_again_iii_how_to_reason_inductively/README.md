# Think Again III: How to Reason Inductively

## Course

This course covers common forms of inductive reasoning, including generalizations, applications of generalizations, inference to the best explanation, analogy, causal reasoning, probability, and decision-making.

## Suggested Reading

Understanding Arguments, Ninth Edition, Concise, Chapters 8-12.

- Welcome to the Course
- Inductive Arguments
- Causal Reasoning
- Chance and Choice
- Catch-Up and Final Quiz
