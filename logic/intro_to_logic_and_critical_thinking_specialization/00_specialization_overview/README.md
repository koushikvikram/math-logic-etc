# Specialization Overview

Duke University's Introduction to Logic and Critical Thinking Specialization is a four-course Coursera sequence focused on recognizing, analyzing, evaluating, and constructing arguments.

## Course Sequence

1. Think Again I: How to Understand Arguments
2. Think Again II: How to Reason Deductively
3. Think Again III: How to Reason Inductively
4. Think Again IV: How to Avoid Fallacies

## Applied Learning Project

The optional final project asks you to construct your own argument on a topic of interest, develop a thesis statement, and write a 400-600 word argument supporting it.
