# Think Again II: How to Reason Deductively

## Course

This course covers deductive validity through propositional logic, truth tables, categorical logic, syllogisms, Venn diagrams, and ways to represent information.

## Suggested Reading

Understanding Arguments, Ninth Edition, Concise, Chapters 6-7.

## Modules

- Welcome to the Course
- Propositional Logic and Truth Tables
- Categorical Logic and Syllogisms
- Representing Information
- Catch-Up and Final Quiz
