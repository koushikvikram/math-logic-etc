# Think Again I: How to Understand Arguments

## Course

This course covers what arguments are, how to identify their parts, how to untangle argumentative language, and how to reconstruct arguments in standard form.

## Suggested Reading

Understanding Arguments, Ninth Edition, Concise, Chapters 1-5.

## Modules

- Welcome to the Specialization
- How to Spot an Argument
- How to Untangle an Argument
- How to Reconstruct an Argument
- Catch-Up and Final Quiz
